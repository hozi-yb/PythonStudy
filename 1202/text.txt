CSS 선택자

기본선택자 : 순수하게 무엇인가를 호출 할 떄
- 전체 선택자
- 태그 선택자
- Class 선택자
- ID 선택자

형태 - 기본선택자
* : 모든 요소를 선택
abc : 태그 이름이 abc인 요소 선택 ex> li
.abc : html class 속성의 값이 abc인 요소 선택 ex> class = "abc"
#abc : html id 속성의 값이 abc인 요소 선택 ex> id = "abc"

복합선택자 : 기본선택자만으로 호출이 힘들 때
- 일치 선택자
- 자식 선택자
- 후손 선택자
- 인접 형제 선택자
- 일반 형제 선택자

형태 - 복합선택자
abc.xyz : 선택자 abc와 xyz를 동시에 만족하는 요소 선택 / 같은 라인에 있을 때.
abc>xyz : 선택자 abc의 자식요소인 xyz를 선택
abc .xyz : 선택자 abc의 하위 요소 xyz 선택. 띄어쓰기가 선택자의 기호이다.
abc+xyz : 선택자 abc의 다음 형제 요소 xyz 하나를 선택
abc~xyz : 선택자 abc의 다음 형제 요소 xyz 모두를 선택

가상선택자 : 사용자의 행동에 따라 변화하는 각 요소의 상황에 따라/ 특정 요소를 부정할 때 사용
- 가상 클래스 선택자
- 사용자의 행동에 따라 변화
- 요소의 상황
- 부정 선택

abc:first-child : 선택자 abc가 형제 요소 중 첫째 일 때, 첫번째 자식을 선택
abc:last-child : 선택자 abc가 형제 요소 중 막내일 때, 마지막 자식을 선택
밑에서 .class가 태그여도 된다.
.class *:nth-child(n) : 클래스내 전체 형제요소중 (n)째라면 선택.
.class *:nth-child(2n) : 클래스 내 전체 형제요소중 짝수자리애들만 선택 (n은 0부터 시작한다.)
.class *:nth-child(2n + 1) : 클래스 내 전체 형제요소중 홀수자리의 애들만 선택
abc:not(xyz) : 선택자 xyz가 아닌 abc 요소 선택

가상요소 : 선택된 요소의 앞, 뒤에 별도에 content를 삽입.
반드시 content라는 속성을 사용
빈 값("")이라도 줘야 적용이 됨.
- after: 요소의 뒤에 내용 삽입
- before : 요소의 앞에 내용 삽입

CSS 선택자 : 지정한 특정 속성을 가지고 있는 태그를 선택하는 선택자
- 특정 속성만 지원
- 속성과 속성의 값을 지정

[disabled] : [ abc ] 대괄호 안에 속성을 포함한 요소 선택../ disabled와 같은
[abc = "xyz"] : 속성 abc를 포함하고 값이 xyz인 요소 선택.
